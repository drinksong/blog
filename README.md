# Introduction

A blog by VuePress

> VuePress generates pre-rendered static HTML for each page, and runs as an SPA once a page is loaded.

[官网](https://vuepress.vuejs.org/)
[中文版](https://vuepress.vuejs.org/zh/)

# Getting started

```
yarn install
```


# Dev

```
yarn run dev
```

# Build

```
yarn run build
```
